艺术没有进化可言，我们读前辈的书（看画、听音乐），应有三种态度：设想在他们的时代鉴赏；据于自己的时代鉴赏；推理未来的时代鉴赏。 举例：希腊雕像（胜利女神），那是三种鉴赏态度都能完全完满肯定。之外，莎士比亚的诗剧、莫扎特的乐曲，也是昔在、今在、永在。
